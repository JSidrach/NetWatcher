CSAJAX_FILTER_DOMAIN

CSAJAX_FILTER_DOMAIN

If set to true, $valid_requests should hold only domains i.e.

a.example.com, b.example.com, usethisdomain.com If set to false, $valid_requests should hold the whole URL ( without the parameters ) i.e. http://example.com/this/is/long/url/ Recommended value: false (for security reasons - do not forget that anyone can access your proxy)
