Proceso del host FPGA:

1) Iniciar con HugePages
  -TODO: como saber si esta activa?
2) ./impact.sh [Fichero de configuracion (bitstream)]
3) Reboot
  - Programada. Para saber que estamos en este paso, `lspci -vvv`, lista los PCI -> sale 1 (xilinx), con id 19aa:e004
4) ./install.sh f
  - Instanciar y montar, solo una vez. Ahora mismo esta con interfaz grafica, cambiar a un script que lo acepte como parametro
  - Para comprobar que esta, `lsmod`, y tiene que salir nfp_driver

5) Para ver si esta en reproductor o capturador
  - `cat /proc/nfp/nfp_report`, ver PLA/REC
  - ./writeControl para controlar cuanto va a enviar/recibir, -l mascara 3

6.1) Recepcion
  - ./configureAel para eliminar ruido
  - ./card2host

6.2) Captura
